Inheritance

Base Classes සහ Derived Classes - කේත නැවත භාවිතා කිරීම සහ විස්තාරණය

Inheritance යනු කුමක්ද?

Inheritance යනු OOP හි එක් ප්‍රධාන සංකල්පයකි. එමගින් නව class එකක් (Derived/Child Class) පවතින class එකක (Base/Parent Class) ගුණාංග (properties) සහ හැසිරීම් (behaviors) උරුම කර ගනී. මෙය කේත නැවත භාවිතා කිරීම (code reusability) සහ class hierarchy නිර්මාණය කිරීමට උපකාරී වේ.

Base Class (Parent)

මූලික ගුණාංග සහ ක්‍රම

Derived Class (Child)

උරුම වූ + නව ගුණාංග සහ ක්‍රම

මූලික Syntax

C++ හි Inheritance භාවිතා කිරීමට මූලික syntax එක:

C++ Inheritance Syntax 
class DerivedClass : access_specifier BaseClass {
    // Derived class members
};

සරල උදාහරණය:

Simple Inheritance Example 
#include <iostream>
using namespace std;

// Base Class
class Animal {
public:
    void eat() {
        cout << "I can eat!" << endl;
    }
    
    void sleep() {
        cout << "I can sleep!" << endl;
    }
};

// Derived Class
class Dog : public Animal {
public:
    void bark() {
        cout << "I can bark! Woof woof!" << endl;
    }
};

int main() {
    Dog myDog;
    
    // Calling base class functions
    myDog.eat();
    myDog.sleep();
    
    // Calling derived class function
    myDog.bark();
    
    return 0;
}
Output:
I can eat!
I can sleep!
I can bark! Woof woof!

Access Specifiers in Inheritance

Inheritance කිරීමේදී access specifiers තුනක් භාවිතා කළ හැක:

Public Inheritance

Base class හි public members, derived class හි public ලෙස පවතී. Protected members, protected ලෙස පවතී.

Protected Inheritance

Base class හි public සහ protected members, derived class හි protected ලෙස පවතී.

Private Inheritance

Base class හි public සහ protected members, derived class හි private ලෙස පවතී.

Access Specifiers Example 
#include <iostream>
using namespace std;

class Base {
public:
    int publicVar;
protected:
    int protectedVar;
private:
    int privateVar;
};

// Public Inheritance
class PublicDerived : public Base {
    // publicVar is public
    // protectedVar is protected
    // privateVar is not accessible
};

// Protected Inheritance
class ProtectedDerived : protected Base {
    // publicVar is protected
    // protectedVar is protected
    // privateVar is not accessible
};

// Private Inheritance
class PrivateDerived : private Base {
    // publicVar is private
    // protectedVar is private
    // privateVar is not accessible
};

Inheritance වර්ග

1. Single Inheritance එක් Base Class

එක් base class එකකින් එක් derived class එකක් නිර්මාණය කිරීම.

Single Inheritance 
#include <iostream>
using namespace std;

class Vehicle {
public:
    string brand = "Ford";
    void honk() {
        cout << "Tuut, tuut!" << endl;
    }
};

class Car : public Vehicle {
public:
    string model = "Mustang";
};

int main() {
    Car myCar;
    myCar.honk();
    cout << myCar.brand + " " + myCar.model << endl;
    return 0;
}

2. Multiple Inheritance කිහිපයක් Base Classes

කිහිපයක් base classes වලින් එක් derived class එකක් නිර්මාණය කිරීම.

Multiple Inheritance 
#include <iostream>
using namespace std;

class Father {
public:
    void showFather() {
        cout << "I am the father" << endl;
    }
};

class Mother {
public:
    void showMother() {
        cout << "I am the mother" << endl;
    }
};

class Child : public Father, public Mother {
public:
    void showChild() {
        cout << "I am the child" << endl;
    }
};

int main() {
    Child obj;
    obj.showFather();
    obj.showMother();
    obj.showChild();
    return 0;
}

3. Multilevel Inheritance තට්ටු කිහිපයක්

Derived class එකක් තවත් base class එකක් බවට පත්වීම.

Multilevel Inheritance 
#include <iostream>
using namespace std;

class Animal {
public:
    void eat() {
        cout << "Eating..." << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "Barking..." << endl;
    }
};

class Puppy : public Dog {
public:
    void weep() {
        cout << "Weeping..." << endl;
    }
};

int main() {
    Puppy myPuppy;
    myPuppy.eat();   // From Animal
    myPuppy.bark();  // From Dog
    myPuppy.weep();  // From Puppy
    return 0;
}

4. Hierarchical Inheritance එක් Base බහු Derived

එක් base class එකකින් කිහිපයක් derived classes නිර්මාණය කිරීම.

Hierarchical Inheritance 
#include <iostream>
using namespace std;

class Shape {
public:
    void display() {
        cout << "This is a shape" << endl;
    }
};

class Circle : public Shape {
public:
    void draw() {
        cout << "Drawing Circle" << endl;
    }
};

class Rectangle : public Shape {
public:
    void draw() {
        cout << "Drawing Rectangle" << endl;
    }
};

int main() {
    Circle c;
    Rectangle r;
    
    c.display();
    c.draw();
    
    r.display();
    r.draw();
    
    return 0;
}

5. Hybrid Inheritance මිශ්‍ර ආකාරය

ඉහත inheritance වර්ග කිහිපයක එකතුවක්.

සටහන: Hybrid inheritance භාවිතා කිරීමේදී "Diamond Problem" ඇති විය හැක. මෙය virtual inheritance භාවිතයෙන් විසඳා ගත හැක.

Inheritance හි Constructors සහ Destructors

Derived class object එකක් නිර්මාණය කරන විට, constructors ක්‍රියාත්මක වන්නේ Base class සිට Derived class දක්වා ය. Destructors ක්‍රියාත්මක වන්නේ ප්‍රතිලෝම අනුපිළිවෙලින් (Derived සිට Base දක්වා).

Constructor/Destructor Order 
#include <iostream>
using namespace std;

class Base {
public:
    Base() {
        cout << "Base Constructor" << endl;
    }
    
    ~Base() {
        cout << "Base Destructor" << endl;
    }
};

class Derived : public Base {
public:
    Derived() {
        cout << "Derived Constructor" << endl;
    }
    
    ~Derived() {
        cout << "Derived Destructor" << endl;
    }
};

int main() {
    cout << "Creating object:" << endl;
    Derived obj;
    cout << "\nDestroying object:" << endl;
    return 0;
}
Output:
Creating object:
Base Constructor
Derived Constructor

Destroying object:
Derived Destructor
Base Destructor

Inheritance වර්ග සංසන්දනය

වර්ගය විස්තරය Base Classes භාවිතය
Single එක් base class, එක් derived class 1 සරල, පොදු භාවිතය
Multiple කිහිපයක් base classes, එක් derived class 2+ කිහිපයක් classes වල විශේෂාංග එකතු කිරීම
Multilevel Chain එකක් ලෙස inheritance 1 (එක් එක් level එකට) ප්‍රධානයෙන් නොමැති දක්වා විශේෂීකරණය
Hierarchical එක් base, කිහිපයක් derived classes 1 පොදු base විවිධ ආකාරවලින්
Hybrid ඉහත වර්ග මිශ්‍රණයක් විවිධ සංකීර්ණ relationships

Inheritance හි ප්‍රතිලාභ

Code Reusability

පවතින කේතය නැවත ලිවීමකින් තොරව භාවිතා කළ හැක.

කේත සංවිධානය

Class hierarchy මගින් කේතය වඩාත් සංවිධානය කළ හැක.

Extensibility

Base class වෙනස් නොකර නව විශේෂාංග එකතු කළ හැක.

Polymorphism

Inheritance මගින් polymorphism ක්‍රියාත්මක කළ හැක.

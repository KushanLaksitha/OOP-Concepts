References in C++

විචල්‍යයන් සඳහා වෙනත් නමක් - Alias for Variables

Reference යනු කුමක්ද?

Reference යනු දැනටමත් පවතින විචල්‍යයක් (variable) සඳහා වෙනත් නමක් වේ. එය විචල්‍යයක alias එකක් ලෙස ක්‍රියා කරයි. Reference එකක් භාවිතා කරන විට, ඔබ original variable එක සමඟ කෙලින්ම වැඩ කරනවා.

ප්‍රධාන ලක්ෂණය: Reference එකක් යනු pointer එකක් මෙන් නොව, එය original variable එකටම තවත් නමක් වේ. Reference එකක් initialize කළ පසු, එය වෙනස් කළ නොහැක.

Syntax

Syntax 
dataType &referenceName = variableName;
සටහන: & (ampersand) සංකේතය භාවිතා කරන්නේ reference එකක් declare කිරීමට.

මූලික උදාහරණය (Basic Example)

C++ 
#include <iostream>
using namespace std;

int main() {
    int num = 10;        // Original variable
    int &ref = num;      // Reference to num
    
    cout << "Original value: " << num << endl;
    cout << "Reference value: " << ref << endl;
    
    // Reference එක හරහා අගය වෙනස් කරන්න
    ref = 20;
    
    cout << "\nපසුව අගය වෙනස් කිරීමෙන්:" << endl;
    cout << "Original value: " << num << endl;
    cout << "Reference value: " << ref << endl;
    
    // Memory address එකම වේ
    cout << "\nMemory Addresses:" << endl;
    cout << "Address of num: " << &num << endl;
    cout << "Address of ref: " << &ref << endl;
    
    return 0;
}
Output:
Original value: 10
Reference value: 10

පසුව අගය වෙනස් කිරීමෙන්:
Original value: 20
Reference value: 20

Memory Addresses:
Address of num: 0x7fff5fbff8ac
Address of ref: 0x7fff5fbff8ac

Memory Visualization

Reference එකක් memory එකේ ක්‍රියා කරන ආකාරය:

num
Value: 10
Address: 0x100
⟷
ref (alias)
Same Value: 10
Same Address: 0x100
වැදගත්: Reference එකක් separate memory location එකක් භාවිතා නොකරයි. එය original variable එකේ තවත් නමක් පමණි.

Functions හි References භාවිතය

References භාවිතයෙන් functions වලට parameters pass කළ හැකිය. මෙය "pass by reference" ලෙස හැඳින්වේ.

Pass by Value (සාමාන්‍ය ක්‍රමය)
Pass by Value 
#include <iostream>
using namespace std;

void increment(int x) {
    x = x + 10;  // මෙය copy එකක් වෙනස් කරයි
}

int main() {
    int num = 5;
    increment(num);
    cout << "Value: " << num << endl;  // Output: 5 (වෙනස් නොවේ)
    return 0;
}
Pass by Reference
Pass by Reference 
#include <iostream>
using namespace std;

void increment(int &x) {  // Reference parameter
    x = x + 10;  // මෙය original එක වෙනස් කරයි
}

int main() {
    int num = 5;
    increment(num);
    cout << "Value: " << num << endl;  // Output: 15 (වෙනස් වේ)
    return 0;
}
වාසි: Pass by reference භාවිතයෙන් large objects copy කිරීම වළක්වා, performance වැඩි දියුණු කළ හැකිය.

Practical Example - Swap Function

References භාවිතයෙන් දෙ අගයන් swap කරන function එකක්:

C++ 
#include <iostream>
using namespace std;

// Reference භාවිතයෙන් swap function
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    
    cout << "Swapping කිරීමට පෙර:" << endl;
    cout << "x = " << x << ", y = " << y << endl;
    
    swap(x, y);
    
    cout << "\nSwapping කිරීමෙන් පසුව:" << endl;
    cout << "x = " << x << ", y = " << y << endl;
    
    return 0;
}
Output:
Swapping කිරීමට පෙර:
x = 10, y = 20

Swapping කිරීමෙන් පසුව:
x = 20, y = 10

Returning References

Functions වලින් references return කළ හැකිය:

C++ 
#include <iostream>
using namespace std;

int globalVar = 100;

// Reference return කරන function
int& getValue() {
    return globalVar;
}

int main() {
    cout << "පළමු අගය: " << getValue() << endl;
    
    // Reference හරහා අගය වෙනස් කරන්න
    getValue() = 200;
    
    cout << "වෙනස් කළ අගය: " << globalVar << endl;
    
    return 0;
}
අවවාදයයි: Local variables වල references return කිරීම භයානක ය! Function එක අවසන් වූ පසු local variable එක destroy වේ.

Const References

වෙනස් කළ නොහැකි references සඳහා const keyword භාවිතා කරන්න:

C++ 
#include <iostream>
#include <string>
using namespace std;

// Const reference - වෙනස් කළ නොහැක
void printName(const string &name) {
    cout << "Name: " << name << endl;
    // name = "New Name";  // ERROR! වෙනස් කළ නොහැක
}

int main() {
    string myName = "Saman";
    printName(myName);
    
    return 0;
}
භාවිතය: Large objects pass කරන විට copy කිරීම වළක්වා, වෙනස් කිරීම ද වළක්වා ගැනීමට const references භාවිතා කරන්න.

References vs Pointers

Feature References (&) Pointers (*)
Null අගය ❌ null විය නොහැක ✅ null විය හැක
Initialization ✅ Declaration කරන විට කළ යුතුය ❌ පසුව කළ හැක
වෙනස් කිරීම ❌ වෙනස් කළ නොහැක ✅ වෙනත් address එකකට point කළ හැක
Syntax Simple - int &ref = x; Complex - int *ptr = &x;
Memory Alias පමණි Separate memory භාවිතා කරයි
භාවිතය Function parameters, return values Dynamic memory, data structures

References භාවිතයේ නීති

1. Initialization අනිවාර්ය: 
int &ref;        // ❌ ERROR - Initialize කළ යුතුය
int x = 10;
int &ref = x;    // ✅ Correct
2. Null විය නොහැක: 
int &ref = NULL;   // ❌ ERROR
int &ref = 0;      // ❌ ERROR
3. වෙනස් කළ නොහැක: 
int x = 10, y = 20;
int &ref = x;
ref = y;    // මෙය y එකට point නොකරයි, x හි අගය y බවට වෙනස් කරයි
4. Array of References නොමැත: 
int &arr[5];   // ❌ ERROR - References array එකක් නැත

Practical Applications

1. Large Objects Efficiently Pass කිරීම
C++ 
#include <iostream>
#include <string>
using namespace std;

class Student {
public:
    string name;
    int age;
    string address;
    
    Student(string n, int a, string addr) {
        name = n;
        age = a;
        address = addr;
    }
};

// Const reference භාවිතයෙන් - copy නොවේ
void displayStudent(const Student &s) {
    cout << "Name: " << s.name << endl;
    cout << "Age: " << s.age << endl;
    cout << "Address: " << s.address << endl;
}

int main() {
    Student student1("Kamal", 20, "Colombo");
    displayStudent(student1);  // Efficient - copy නොවේ
    
    return 0;
}
2. Multiple Values Return කිරීම
C++ 
#include <iostream>
using namespace std;

// References භාවිතයෙන් multiple values return කරන්න
void getMinMax(int arr[], int size, int &min, int &max) {
    min = max = arr[0];
    
    for(int i = 1; i < size; i++) {
        if(arr[i] < min) min = arr[i];
        if(arr[i] > max) max = arr[i];
    }
}

int main() {
    int numbers[] = {45, 23, 78, 12, 90, 34};
    int minimum, maximum;
    
    getMinMax(numbers, 6, minimum, maximum);
    
    cout << "Minimum: " << minimum << endl;
    cout << "Maximum: " << maximum << endl;
    
    return 0;
}
3. Operator Overloading
C++ 
#include <iostream>
using namespace std;

class Counter {
private:
    int count;
public:
    Counter() : count(0) {}
    
    // Prefix ++ operator - reference return කරයි
    Counter& operator++() {
        count++;
        return *this;
    }
    
    void display() {
        cout << "Count: " << count << endl;
    }
};

int main() {
    Counter c;
    ++c;
    ++c;
    c.display();  // Output: Count: 2
    
    return 0;
}

සාමාන්‍ය වැරදි (Common Mistakes)

❌ වැරදි 1: Local Variable හි Reference Return කිරීම 
int& getNumber() {
    int num = 100;  // Local variable
    return num;     // ❌ භයානකයි! num destroy වේ
}

int main() {
    int &ref = getNumber();  // Undefined behavior
    cout << ref << endl;      // Crash විය හැක
}
❌ වැරදි 2: Uninitialized Reference 
int &ref;  // ❌ ERROR - Compile error
// References initialize කළ යුතුය
❌ වැරදි 3: Reference to Temporary 
int& ref = 10;  // ❌ ERROR
// Temporary values වලට non-const reference එකක් නොමැත

const int& ref = 10;  // ✅ Correct - const reference OK

Advanced Example - Matrix Class

References භාවිතයෙන් complex program එකක්:

C++ 
#include <iostream>
using namespace std;

class Matrix {
private:
    int data[3][3];
public:
    Matrix() {
        for(int i = 0; i < 3; i++)
            for(int j = 0; j < 3; j++)
                data[i][j] = 0;
    }
    
    // Reference return කරන accessor
    int& at(int row, int col) {
        return data[row][col];
    }
    
    // Const reference parameter
    void add(const Matrix &other) {
        for(int i = 0; i < 3; i++)
            for(int j = 0; j < 3; j++)
                data[i][j] += other.data[i][j];
    }
    
    void display() const {
        for(int i = 0; i < 3; i++) {
            for(int j = 0; j < 3; j++)
                cout << data[i][j] << " ";
            cout << endl;
        }
    }
};

int main() {
    Matrix mat1, mat2;
    
    // Reference භාවිතයෙන් values set කරන්න
    mat1.at(0, 0) = 1;
    mat1.at(1, 1) = 2;
    mat1.at(2, 2) = 3;
    
    mat2.at(0, 0) = 10;
    mat2.at(1, 1) = 20;
    mat2.at(2, 2) = 30;
    
    cout << "Matrix 1:" << endl;
    mat1.display();
    
    cout << "\nMatrix 2:" << endl;
    mat2.display();
    
    mat1.add(mat2);
    
    cout << "\nපසුව Matrix 1 + Matrix 2:" << endl;
    mat1.display();
    
    return 0;
}
Output:
Matrix 1:
1 0 0
0 2 0
0 0 3

Matrix 2:
10 0 0
0 20 0
0 0 30

පසුව Matrix 1 + Matrix 2:
11 0 0
0 22 0
0 0 33

සාරාංශය (Summary)

  • Reference යනු variable එකක් සඳහා වෙනත් නමක් (alias) වේ
  • & සංකේතය භාවිතයෙන් references declare කරයි
  • References initialize කළ යුතු අතර null විය නොහැක
  • Initialize කළ පසු reference වෙනස් කළ නොහැක
  • Pass by reference භාවිතයෙන් functions වලට efficient ලෙස parameters pass කරයි
  • Const references භාවිතයෙන් large objects efficiently pass කරන අතර වෙනස් කිරීම වළක්වයි
  • Local variables වල references return කිරීම භයානකයි
  • Pointers වලට වඩා references භාවිතය සරල වේ

මතක තබා ගත යුතු දේවල්

භාවිතා කරන්න References:
  • Function parameters සඳහා (විශේෂයෙන් large objects)
  • Operator overloading සඳහා
  • Multiple return values සඳහා
  • සරල සහ safe syntax අවශ්‍ය විට
භාවිතා නොකරන්න References:
  • Null values අවශ්‍ය විට (pointers භාවිතා කරන්න)
  • Reassignment අවශ්‍ය විට (pointers භාවිතා කරන්න)
  • Dynamic memory management සඳහා (pointers භාවිතා කරන්න)
  • Arrays of references අවශ්‍ය විට (vectors භාවිතා කරන්න)

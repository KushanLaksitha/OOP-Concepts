Inheritance

Code නැවත භාවිතා කිරීමේ බලය (The Power of Reusability)

Inheritance යනු කුමක්ද?

Inheritance යනු එක් class එකක (derived/child class) ගුණාංග (attributes) සහ ක්‍රියා (methods) තවත් class එකකින් (base/parent class) ලබා ගැනීමේ ක්‍රියාවලියයි. මෙය "is-a" සම්බන්ධතාවයක් නිරූපණය කරයි (e.g., a Car "is a" Vehicle). Inheritance හි ප්‍රධානම අරමුණ වන්නේ code reusability (කේත නැවත භාවිතා කිරීමේ හැකියාව) වැඩි කිරීමයි.

Base Class (Parent): වෙනත් class එකකට තම ගුණාංග ලබා දෙන class එක.
Derived Class (Child): Base class එකෙන් ගුණාංග ලබා ගන්නා class එක. Derived class එකට Base class එකේ ගුණාංග වලට අමතරව තමන්ටම ආවේණික ගුණාංග සහ ක්‍රියා ද තිබිය හැක.

සැබෑ ලෝක උදාහරණය: වාහන (Vehicles)

`Vehicle` (වාහනය) නමින් මූලික class එකක් සිතන්න. ඕනෑම වාහනයකට brand එකක්, model එකක්, සහ `start()`, `stop()` වැනි පොදු ක්‍රියාකාරකම් ඇත.

දැන් `Car`, `Motorcycle`, `Truck` වැනි derived classes සෑදිය හැක. මෙම සියලු classes, `Vehicle` class එකෙන් brand, model, start(), stop() වැනි පොදු ගුණාංග උරුම කර ගනී. ඉන්පසු, `Car` class එකට `openTrunk()` වැනි එයටම ආවේණික method එකක්ද, `Motorcycle` class එකට `doWheelie()` වැනි method එකක්ද එකතු කළ හැක. `Vehicle` class එකේ code එක නැවත නැවත ලිවීම අවශ්‍ය නොවේ.

C++ හි Inheritance ක්‍රියාත්මක කිරීම

C++ හිදී, derived class එකක් නිර්වචනය කිරීමේදී `:` (colon) සලකුණ භාවිතා කරනු ලැබේ. මෙහිදී `protected` access specifier එක වැදගත් වේ. `protected` members, base class එකට සහ එහි derived classes වලට පමණක් ප්‍රවේශ විය හැක, නමුත් class එකෙන් පිටතදී ප්‍රවේශ විය නොහැක.

C++ 
#include <iostream>
#include 
using namespace std;
// Base Class (Parent)
class Person {
protected: // Can be accessed by derived classes
string name;
int age;
public:
Person(string n = "", int a = 0) {
name = n;
age = a;
}
void introduce() {
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
}
};
// Derived Class (Child)
// Syntax: class DerivedClass : access_specifier BaseClass
class Student : public Person {
private: // Specific to Student
string studentID;
public:
// Call the base class constructor
Student(string n, int a, string id) : Person(n, a) {
studentID = id;
}
// New method specific to Student
void study() {
    cout << name << " is studying." << endl;
}

// Overriding the introduce method (optional, will be covered in Polymorphism)
void introduceStudent() {
    cout << "--- Student Details ---" << endl;
    introduce(); // Call base class method
    cout << "Student ID: " << studentID << endl;
}
};
int main() {
// Create a Person object
Person teacher("Mr. Silva", 45);
cout << "--- Teacher (Base Class Object) ---" << endl;
teacher.introduce();
cout << "\n----------------------------------\n" << endl;

// Create a Student object
Student student1("Nimal", 21, "S12345");
student1.introduceStudent(); // Accessing derived class method
student1.study();            // Accessing derived class method

// The student object can also access the public methods of the Person class
// student1.introduce(); // This would work too

return 0;
}
Output:
--- Teacher (Base Class Object) ---
Name: Mr. Silva
Age: 45

----------------------------------

--- Student Details ---
Name: Nimal
Age: 21
Student ID: S12345
Nimal is studying.

Inheritance වර්ග (Types of Inheritance)

OOP වල Inheritance ප්‍රධාන වර්ග කිහිපයක් ඇත:

Single

එක් derived class එකක්, එක් base class එකකින් උරුම වීම.

Multiple

එක් derived class එකක්, base classes කිහිපයකින් උරුම වීම.

Multilevel

Derived class එකක් තවත් derived class එකකින් උරුම වීම.

Hierarchical

එකම base class එකකින් derived classes කිහිපයක් උරුම වීම.

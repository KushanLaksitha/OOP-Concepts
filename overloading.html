Overloading vs Overriding

Compile-time සහ Run-time Polymorphism අතර වෙනස

Function Overloading (Compile-time)

Function Overloading යනු එකම class එකක් තුළ, එකම නම සහිත functions කිහිපයක්, ඒවායේ parameters (arguments) ගණනින් හෝ වර්ගයෙන් වෙනස් වන පරිදි නිර්වචනය කිරීමයි. Compiler එක විසින් call කරන අවස්ථාවේදී ලබාදෙන arguments වලට ගැලපෙන function එක තෝරා ගනී.

C++ Overloading 
#include <iostream>
using namespace std;

class Calculator {
public:
    // එකම නම, විවිධ parameters
    int add(int a, int b) {
        cout << "Adding two integers: ";
        return a + b;
    }
    
    double add(double a, double b) {
        cout << "Adding two doubles: ";
        return a + b;
    }
    
    int add(int a, int b, int c) {
        cout << "Adding three integers: ";
        return a + b + c;
    }
};

int main() {
    Calculator calc;
    
    cout << calc.add(5, 3) << endl;        // Calls the first function
    cout << calc.add(5.5, 3.2) << endl;  // Calls the second function
    cout << calc.add(5, 3, 2) << endl;     // Calls the third function
    
    return 0;
}
Output:
Adding two integers: 8
Adding two doubles: 8.7
Adding three integers: 10

Method Overriding (Run-time)

Method Overriding යනු derived class එකක් තුළ, base class එකේ දැනටමත් පවතින function එකක් (එකම නම, එකම parameters සහිත) නැවත අලුතින් නිර්වචනය කිරීමයි (re-implementation). මෙය Inheritance සමඟ භාවිතා වන අතර, `virtual` keyword එක හරහා Run-time Polymorphism ලබා ගැනීමට උපකාරී වේ.

C++ Overriding 
#include <iostream>
using namespace std;

class Base {
public:
    virtual void show() { // Virtual function in the base class
        cout << "This is the Base class show() function." << endl;
    }
};

class Derived : public Base {
public:
    void show() override { // Re-implementing (overriding) the function
        cout << "This is the Derived class show() function." << endl;
    }
};

int main() {
    Base* b_ptr;    // Base class pointer
    Derived d_obj;  // Derived class object
    
    b_ptr = &d_obj;
    
    // Calls the Derived class's show() function because of 'virtual'
    b_ptr->show(); 
    
    return 0;
}
Output:
This is the Derived class show() function.

සාරාංශගත සැසඳීම

Feature Function Overloading Method Overriding
අරමුණ (Purpose) එකම නම සහිත functions වලට විවිධ කාර්යයන් ලබා දීම. Base class function එකට derived class එකේදී විශේෂිත implementation එකක් ලබා දීම.
Class Scope එකම class එකක් තුළ සිදු වේ. Base සහ Derived classes දෙකක් අතර සිදු වේ.
Function Signature Parameters වෙනස් විය යුතුය (Must be different). Parameters සහ නම සමාන විය යුතුය (Must be the same).
Inheritance අවශ්‍ය නොවේ (). අනිවාර්යයෙන්ම අවශ්‍ය වේ ().
Polymorphism Type Compile-time (Static) Run-time (Dynamic)
Mechanism Compiler විසින් arguments අනුව function තෝරා ගනී. `virtual` keyword එක මගින් object type එක අනුව run-time හිදී function තෝරා ගනී.

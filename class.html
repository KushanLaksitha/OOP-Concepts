Classes & Objects

OOP හි මූලික ගොඩනැඟිලි කොටස් (The Core Building Blocks)

Class යනු කුමක්ද?

Class එකක් යනු object එකක් සෑදීමට අවශ්‍ය blueprint (සැලැස්ම) එකකි. එය object එකක තිබිය යුතු ගුණාංග (attributes) සහ ක්‍රියා (methods) මොනවාදැයි නිර්වචනය කරයි. සරලවම, class එකක් යනු අප විසින්ම සාදාගන්නා custom data type එකකි.

Attributes: Class එකක් තුළ ඇති variables වේ. මේවා object එකේ data (තොරතුරු) ගබඩා කර තබා ගනී. (e.g., car එකක පාට, brand එක).
Methods: Class එකක් තුළ ඇති functions වේ. මේවා object එකට කළ හැකි ක්‍රියාවන් නිරූපණය කරයි. (e.g., car එකක් drive කිරීම, brake කිරීම).

සැබෑ ලෝක උදාහරණය (Real-World Analogy)

නිවසක් තැනීමේ සැලැස්මක් (blueprint) class එකක් ලෙස සිතන්න. එම සැලැස්මේ නිවසේ කාමර ගණන, ජනේල, දොරවල් වැනි විස්තර (attributes) සහ දොරවල් ඇරීම, වැසීම වැනි ක්‍රියා (methods) අඩංගු වේ.

මෙම සැලැස්ම භාවිතා කර තනන ලද සැබෑ නිවාස objects වේ. එකම සැලැස්මෙන් නිවාස කිහිපයක් සෑදිය හැකි සේම, එකම class එකෙන් objects කිහිපයක් සෑදිය හැක. සෑම නිවසකටම (object) එයටම ආවේණික වර්ණයක් හෝ ලිපිනයක් (attribute values) තිබිය හැක.

Class එකක් Define කරන්නේ කෙසේද?

C++ වල `class` keyword එක භාවිතා කර class එකක් නිර්වචනය කරයි. Class එකේ body එක `{}` curly brackets තුළ ලියන අතර අවසානයේ semicolon (`;`) එකක් යොදනු ලැබේ.

C++ 
#include <iostream>
#include 
using namespace std;
// Student class එක define කිරීම
class Student {
public: // Access Specifier
// Attributes
string name;
int age;
string studentID;
code
Code
// Method
void introduce() {
    cout << "Hi, I'm " << name << " and I am " << age << " years old." << endl;
    cout << "My Student ID is: " << studentID << endl;
}
};
int main() {
// ------------------------------------------
// Object එකක් සාදා ගැනීම (Instantiation)
// ------------------------------------------
Student student1; // student1 නමින් object එකක් create කිරීම
code
Code
// ------------------------------------------
// Attributes වලට values assign කිරීම
// ------------------------------------------
student1.name = "Nimal";
student1.age = 20;
student1.studentID = "S12345";

// ------------------------------------------
// Method එක call කිරීම
// ------------------------------------------
student1.introduce();

cout << "\n--- Another Object ---\n" << endl;

// තවත් object එකක් create කිරීම
Student student2;
student2.name = "Kamala";
student2.age = 22;
student2.studentID = "S67890";
student2.introduce();

return 0;
}
Output:
Hi, I'm Nimal and I am 20 years old.
My Student ID is: S12345

--- Another Object ---

Hi, I'm Kamala and I am 22 years old.
My Student ID is: S67890
Access Specifiers: `public` යනු access specifier එකකි. Public ලෙස declare කළ members class එකෙන් පිටතදී access කළ හැක. `private` සහ `protected` ලෙස තවත් specifiers දෙකක් ඇත, ඒවා data security සඳහා වැදගත් වේ.

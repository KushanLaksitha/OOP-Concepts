පාඩම 1: Activity Diagram හැඳින්වීම

Activity Diagram යනු කුමක්ද?

Activity Diagram එකක් යනු program එකක flow එක (ක්‍රියාවලි අනුපිළිවෙල) පෙන්වන diagram එකකි. මේකෙන් code එක ලියන්න කලින් program එක සැලසුම් කරන්න පුළුවන්.

මූලික Node වර්ග:

1. Start Node (ආරම්භය) ⚫
Start

Program එක ආරම්භ වන තැන. සෑම program එකකම මෙය පළමුව තිබිය යුතුයි.

2. End Node (අවසානය) ⊗
End

Program එක අවසන් වන තැන.

පාඩම 2: Process Statements

Process Node යනු?

Process node එකෙන් සාමාන්‍ය statements ලියන්න පුළුවන්. උදාහරණ: variables declare කිරීම, calculations, assignments වගේ දේවල්.

පාඩම 3: Input සහ Output

Input/Output යනු?

Input nodes වලින් user input ගන්නවා. Output nodes වලින් results display කරනවා.

1. Input Node (ආදානය) ⬇
age

User input එකක් ගන්න. C++ වලදී cin භාවිතා වෙයි.

Code: cin >> age;

2. Output Node (ප්‍රතිදානය) ⬆
result

Output එකක් display කරන්න. C++ වලදී cout භාවිතා වෙයි.

Code: cout << result << endl;

උදාහරණය: සරල Calculator
Activity Diagram:

↓

num2
↓

C++ Code:
#include <iostream>
using namespace std;
int main() {
int num1, num2;
// Variables must be declared!
cout << "Enter two numbers: ";
cin >> num1 >> num2;
int sum = num1 + num2;
cout << "Sum is: ";
cout << sum << endl;
return 0;
}
වැදගත් උපදෙස්:
  • Input: Variable name එක පමණක් යොදන්න.
  • Output: Variable name එක හෝ string message එකක් (" ") දාන්න.
  • Variables: Input ගන්නා variables code එකේ declare කරන්න අමතක කරන්න එපා.

පාඩම 4: Decisions (if/else)

Decision Node යනු?

Decision node එකෙන් conditions check කරන්න පුළුවන්. මේක C++ වල if/else statement එකකට convert වෙනවා. Decision එකකට TRUE සහ FALSE branches දෙකක් තියෙනවා.

Decision Node ◇
age >= 18

Diamond shape එකක condition ලියනවා. මෙයින් TRUE සහ FALSE ලෙස මාර්ග දෙකක් විහිදේ.

උදාහරණය: වයස check කිරීම

↓

C++ Code:
#include <iostream>
using namespace std;
int main() {
int age;
cin >> age;
if (age >= 18) {
cout << "Can vote" << endl;
} else {
cout << "Cannot vote" << endl;
}
return 0;
}
භාවිතා කළ හැකි Conditions:
  • == - සමාන
  • != - සමාන නොවේ
  • > - විශාල
  • < - කුඩා
  • >= - විශාල හෝ සමාන
  • <= - කුඩා හෝ සමාන
පුහුණු අභ්‍යාස:
  1. ධන හෝ ඍණ check කරන program එකක්
  2. ලකුණ 50ට වැඩි නම් "Pass" නැත්නම් "Fail" print කරන program එකක්

පාඩම 5: සම්පූර්ණ උදාහරණ

සම්පූර්ණ Program එකක්

දැන් අපි ඉගෙන ගත්ත සියලු දේවල් එකතු කරලා සම්පූර්ණ program එකක් හදමු.

උදාහරණය: ලකුණු සාමාන්‍යය ගණනය කිරීම

අවශ්‍යතාව: විෂය 3කට ලකුණු ගෙන average එක calculate කරලා pass/fail check කරන්න (pass mark = 40)

Activity Diagram:
Start

mark1
↓

සම්පූර්ණ C++ Code:
#include <iostream>
using namespace std;
int main() {
// Declare variables
float mark1, mark2, mark3;
// Input marks
cout << "Enter 3 marks: ";
cin >> mark1 >> mark2 >> mark3;
// Calculate average
float avg = (mark1 + mark2 + mark3) / 3.0;
// Display average
cout << "Average: " << avg << endl;
// Check pass or fail
if (avg >= 40) {
cout << "Pass" << endl;
} else {
cout << "Fail" << endl;
}
return 0;
}

සාරාංශය

Node වර්ග
  • ⚫ Start - ආරම්භය
  • ⊗ End - අවසානය
  • ▭ Process - ක්‍රියාවලිය
  • ◇ Decision - තීරණය
  • ⬇ Input - ආදානය
  • ⬆ Output - ප්‍රතිදානය
C++ Conversions
  • Process: variable = value;
  • Input: cin >> variable;
  • Output: cout << value;
  • Decision: if (condition) { } else { }
ප්‍රධාන කරුණු
  • සෑම program එකකම Start/End තියෙන්න ඕන.
  • Variables declare කරන්න මතක තියාගන්න.
  • Decision වලට TRUE/FALSE branches දෙකම දාන්න.
  • Flow එක logical order එකේ තියාගන්න.

Polymorphism

එකම ක්‍රියාව, විවිධ రూప (One Action, Many Forms)

Polymorphism යනු කුමක්ද?

Polymorphism යනු ග්‍රීක වචනයක් වන අතර එහි තේරුම "බොහෝ രൂപ" (many forms) යන්නයි. OOP හිදී, Polymorphism යනු එකම නමක් ඇති task එකක් (method/operator) විවිධ ආකාරවලින් සිදු කිරීමේ හැකියාවයි. Inheritance සංකල්පය සමඟ මෙය ඉතා බලවත් ලෙස ක්‍රියාත්මක වේ.

සැබෑ ලෝක උදාහරණය: ඔබ

ඔබම Polymorphism සඳහා හොඳ උදාහරණයකි. ඔබ එකම පුද්ගලයා වුවත්, විවිධ අවස්ථාවලදී විවිධ ලෙස හැසිරේ.

  • පාසලේදී: ඔබ ශිෂ්‍යයෙක්.
  • නිවසේදී: ඔබ දරුවෙක්.
  • මිතුරන් සමඟ: ඔබ මිතුරෙක්.
ඔබ එකම පුද්ගලයා වුවත් (one person), ඔබගේ හැසිරීම (action) අවස්ථාව අනුව වෙනස් වේ (many forms).

C++ හි Polymorphism වර්ග

C++ හි Polymorphism ප්‍රධාන වර්ග දෙකකි:

1. Compile-time (Static) Polymorphism: මෙහිදී කුමන function එක call කළ යුතුද යන්න compile time එකේදීම තීරණය වේ. මෙය Function Overloading සහ Operator Overloading මගින් ලබා ගනී.
2. Run-time (Dynamic) Polymorphism: මෙහිදී කුමන function එක call කළ යුතුද යන්න program එක run වන විට තීරණය වේ. මෙය Virtual Functions සහ Function Overriding මගින් ලබා ගනී.

උදාහරණය: Run-time Polymorphism (Function Overriding)

Run-time Polymorphism ලබා ගැනීමට, base class එකේ function එක `virtual` keyword එක සමඟ declare කළ යුතුය. ඉන්පසු derived class එකේ එම function එකම නැවත implement (override) කරනු ලැබේ.

C++ 
#include <iostream>
using namespace std;

// Base Class
class Animal {
public:
    // The 'virtual' keyword enables dynamic polymorphism
    virtual void makeSound() {
        cout << "The animal makes a sound." << endl;
    }
};

// Derived Class 1
class Dog : public Animal {
public:
    // Overriding the base class function
    void makeSound() {
        cout << "The dog barks: Woof! Woof!" << endl;
    }
};

// Derived Class 2
class Cat : public Animal {
public:
    // Overriding the base class function
    void makeSound() {
        cout << "The cat meows: Meow! Meow!" << endl;
    }
};

int main() {
    Animal* myAnimal; // A pointer to the base class
    
    Dog myDog;
    Cat myCat;
    
    // Point to a Dog object
    myAnimal = &myDog;
    myAnimal->makeSound(); // Calls the Dog's makeSound() method

    // Point to a Cat object
    myAnimal = &myCat;
    myAnimal->makeSound(); // Calls the Cat's makeSound() method
    
    return 0;
}
Output:
The dog barks: Woof! Woof!
The cat meows: Meow! Meow!
මෙහිදී වැදගත් වන්නේ:
  • `myAnimal` යනු `Animal` class එකේ pointer එකක් වුවත්, එය `Dog` object එකකට point කරන විට `Dog` class එකේ `makeSound()` method එකත්, `Cat` object එකකට point කරන විට `Cat` class එකේ method එකත් call වීමයි.
  • මෙම තීරණය program එක run වන විට (run-time) ගන්නා නිසා මෙය dynamic polymorphism ලෙස හැඳින්වේ.
  • `virtual` keyword එක නොතිබුනේ නම්, සෑම විටම `Animal` class එකේ `makeSound()` method එක පමණක් call වනු ඇත.

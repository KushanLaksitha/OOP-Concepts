Structures (struct)

C++ හි Structures භාවිතා කරන ආකාරය ඉගෙන ගනිමු

Structure යනු කුමක්ද?

Structure (struct) යනු විවිධ data types එකට එක් unit එකක් ලෙස group කිරීමට භාවිතා කරන user-defined data type එකකි. මෙය classes වලට සමාන නමුත් වෙනස්කම් කිහිපයක් ඇත.

ප්‍රධාන ලක්ෂණ: Structures හි සියලුම members default වශයෙන් public වේ. Classes වලදී default වශයෙන් private වේ.

මූලික Syntax

C++ 
struct StructureName {
    // Data members
    dataType member1;
    dataType member2;
    // ...
    
    // Member functions (optional)
    void functionName() {
        // function body
    }
};

සරල උදාහරණය - Student Record

C++ 
#include <iostream>
#include <string>
using namespace std;

// Student structure එක define කිරීම
struct Student {
    int id;
    string name;
    float gpa;
    
    // Member function
    void display() {
        cout << "ID: " << id << endl;
        cout << "Name: " << name << endl;
        cout << "GPA: " << gpa << endl;
    }
};

int main() {
    // Structure variable එකක් create කිරීම
    Student s1;
    
    // Members වලට values assign කිරීම
    s1.id = 101;
    s1.name = "Kamal Perera";
    s1.gpa = 3.75;
    
    // Display function එක call කිරීම
    s1.display();
    
    return 0;
}
Output:
ID: 101
Name: Kamal Perera
GPA: 3.75

Structures Initialize කිරීමේ ක්‍රම

1. Member-wise Initialization
C++ 
struct Point {
    int x;
    int y;
};

Point p1;
p1.x = 10;
p1.y = 20;
2. Aggregate Initialization
C++ 
struct Point {
    int x;
    int y;
};

Point p1 = {10, 20};  // සරලම ක්‍රමය
Point p2{15, 25};     // C++11 style
3. Constructor සමඟ
C++ 
struct Point {
    int x;
    int y;
    
    // Constructor
    Point(int xVal, int yVal) {
        x = xVal;
        y = yVal;
    }
};

Point p1(10, 20);

Nested Structures

Structure එකක් තුළ තවත් structure එකක් define කළ හැකිය:

C++ 
#include <iostream>
#include <string>
using namespace std;

struct Address {
    string street;
    string city;
    int zipCode;
};

struct Employee {
    int id;
    string name;
    Address address;  // Nested structure
    
    void display() {
        cout << "ID: " << id << endl;
        cout << "Name: " << name << endl;
        cout << "Address: " << address.street << ", " 
             << address.city << " - " << address.zipCode << endl;
    }
};

int main() {
    Employee emp;
    emp.id = 201;
    emp.name = "Nimal Silva";
    emp.address.street = "Main Street";
    emp.address.city = "Colombo";
    emp.address.zipCode = 10100;
    
    emp.display();
    
    return 0;
}

Array of Structures

Structure objects ගණනාවක් array එකක store කළ හැකිය:

C++ 
#include <iostream>
#include <string>
using namespace std;

struct Book {
    string title;
    string author;
    float price;
};

int main() {
    // Array of structures
    Book library[3] = {
        {"C++ Programming", "Bjarne Stroustrup", 1500.00},
        {"Data Structures", "Seymour Lipschutz", 1200.00},
        {"Algorithms", "Thomas Cormen", 2000.00}
    };
    
    // Display all books
    cout << "Library Books:" << endl;
    cout << "------------------------" << endl;
    for(int i = 0; i < 3; i++) {
        cout << "Title: " << library[i].title << endl;
        cout << "Author: " << library[i].author << endl;
        cout << "Price: Rs. " << library[i].price << endl;
        cout << "------------------------" << endl;
    }
    
    return 0;
}

Pointer to Structure

C++ 
#include <iostream>
#include <string>
using namespace std;

struct Car {
    string brand;
    string model;
    int year;
};

int main() {
    Car myCar = {"Toyota", "Corolla", 2023};
    
    // Structure pointer එකක් create කිරීම
    Car *ptr = &myCar;
    
    // Arrow operator (->) භාවිතයෙන් access කිරීම
    cout << "Brand: " << ptr->brand << endl;
    cout << "Model: " << ptr->model << endl;
    cout << "Year: " << ptr->year << endl;
    
    return 0;
}
සටහන: Pointer භාවිතයෙන් structure members access කිරීමට arrow operator (->) භාවිතා කරන්න. Dot operator (.) normal variables සඳහා භාවිතා වේ.

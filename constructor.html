Constructors & Destructors

Object-Oriented Programming හි Object ජීවන චක්‍රය කළමනාකරණය

Constructors

Constructor යනු class එකක object එකක් නිර්මාණය කරන විට ස්වයංක්‍රීයව ක්‍රියාත්මක වන විශේෂ member function එකකි. එහි ප්‍රධාන අරමුණ වන්නේ object එකේ data members වලට ආරම්භක අගයන් ලබා දීමයි (initialization). Constructor එකේ නම class එකේ නමට සමාන විය යුතු අතර, එයට return type එකක් නොමැත.

Constructor වර්ග:

1. Default Constructor

Parameters කිසිවක් නොගන්නා constructor එක Default Constructor ලෙස හැඳින්වේ.

C++ Default Constructor 
#include <iostream>
using namespace std;

class Wall {
private:
    double length;
public:
    // Default constructor
    Wall() {
        length = 5.5;
        cout << "Default constructor called. Length set to " << length << endl;
    }
};

int main() {
    Wall wall1; // Calls the default constructor
    return 0;
}
Output:
Default constructor called. Length set to 5.5
2. Parameterized Constructor

Parameters ලබා ගන්නා constructor, Parameterized Constructor ලෙස හැඳින්වේ. Object එකක් නිර්මාණය කිරීමේදී අගයන් ලබා දීමට මෙය භාවිතා කරයි.

C++ Parameterized Constructor 
#include <iostream>
using namespace std;

class Wall {
private:
    double length;
    double height;
public:
    // Parameterized constructor
    Wall(double len, double hgt) {
        length = len;
        height = hgt;
        cout << "Parameterized constructor called. Area is " << length * height << endl;
    }
};

int main() {
    Wall wall1(10.5, 8.6); // Calls the parameterized constructor
    return 0;
}
Output:
Parameterized constructor called. Area is 90.3
3. Copy Constructor

දැනටමත් පවතින object එකක අගයන් භාවිතා කරමින් නව object එකක් initialize කිරීමට Copy Constructor භාවිතා වේ.

C++ Copy Constructor 
#include <iostream>
using namespace std;

class Point {
private:
    int x, y;
public:
    Point(int x1, int y1) { x = x1; y = y1; }
    
    // Copy constructor
    Point(const Point &p2) {
        x = p2.x; 
        y = p2.y; 
    }

    int getX() { return x; }
    int getY() { return y; }
};

int main() {
    Point p1(10, 15); // Normal constructor
    Point p2 = p1; // Copy constructor called here
    cout << "p1.x = " << p1.getX() << ", p1.y = " << p1.getY() << endl;
    cout << "p2.x = " << p2.getX() << ", p2.y = " << p2.getY() << endl;
    return 0;
}
Output:
p1.x = 10, p1.y = 15
p2.x = 10, p2.y = 15

Destructors

Destructor යනු object එකක් විනාශ වන විට (උදා: function එකක scope එකෙන් ඉවත් වන විට හෝ `delete` keyword එක භාවිතා කරන විට) ස්වයංක්‍රීයව ක්‍රියාත්මක වන විශේෂ member function එකකි. එහි ප්‍රධාන අරමුණ වන්නේ object එක විසින් වෙන් කර ගන්නා ලද සම්පත් (resources) නිදහස් කිරීමයි (උදා: dynamically allocated memory). Destructor එකේ නම class නමට සමාන වන අතර, එයට පෙර tilde (~) ලකුණ යොදනු ලැබේ. එයට parameters හෝ return type එකක් නොමැත.

C++ Destructor 
#include <iostream>
using namespace std;

class MyClass {
public:
    // Constructor
    MyClass() {
        cout << "Constructor called: Object created." << endl;
    }
    
    // Destructor
    ~MyClass() {
        cout << "Destructor called: Object destroyed." << endl;
    }
};

int main() {
    cout << "Entering main function." << endl;
    MyClass obj1; // Object created, constructor called
    cout << "Exiting main function." << endl;
    // obj1 goes out of scope here, destructor is called
    return 0;
}
Output:
Entering main function.
Constructor called: Object created.
Exiting main function.
Destructor called: Object destroyed.

Constructor vs Destructor

Feature Constructor Destructor
Purpose Object එකක data members initialize කිරීම. Object එක විසින් භාවිතා කළ සම්පත් නිදහස් කිරීම.
Name Class නමට සමාන වේ (e.g., `MyClass()`). Tilde (~) ලකුණ සහ Class නම (e.g., `~MyClass()`).
Called When Object එකක් නිර්මාණය කරන විට. Object එකක් විනාශ වන විට.
Parameters Parameters තිබිය හැක (Overloaded විය හැක). Parameters නොමැත (Overloaded විය නොහැක).
Return Type Return type එකක් නොමැත. Return type එකක් නොමැත.

Object Oriented Programming

C++ යොදාගෙන OOP සංකල්ප ඉගෙන ගනිමු

OOP යනු කුමක්ද?

Object Oriented Programming (OOP) යනු programming paradigm එකක් වන අතර එය objects සහ classes මත පදනම් වී ඇත. මෙය code reusability සහ modularity වැඩි කිරීමට උපකාරී වේ.

Class: Objects සාදා ගැනීම සඳහා භාවිතා කරන blueprint එකකි. එය attributes සහ methods අඩංගු වේ.
Object: Class එකක instance එකකි. එය specific data සහිතව පවතී.

ප්‍රධාන සංකල්ප (Key Concepts)

Encapsulation

Data සහ methods එක class එකක් තුළ එකතු කිරීම

Abstraction

අනවශ්‍ය details සඟවා ගැනීම

Inheritance

එක් class එකක properties වෙනත් class එකකට ලබා ගැනීම

Polymorphism

එකම interface එකක් විවිධ ආකාරවලින් භාවිතා කිරීම

සරල උදාහරණයක් (Simple Example)

මෙන්න සරල Car class එකක උදාහරණයක්:

C++ 
#include <iostream>
#include <string>
using namespace std;

// Car class එක define කිරීම
class Car {
    // Private members - encapsulation
    private:
        string brand;
        string model;
        int year;
    
    // Public members
    public:
        // Constructor - initialize කිරීමට
        Car(string b, string m, int y) {
            brand = b;
            model = m;
            year = y;
        }
        
        // Method - කාර් එකේ විස්තර display කිරීමට
        void displayInfo() {
            cout << "Brand: " << brand << endl;
            cout << "Model: " << model << endl;
            cout << "Year: " << year << endl;
        }
        
        // Method - කාර් එක drive කිරීමට
        void drive() {
            cout << brand << " " << model << " is driving!" << endl;
        }
};

int main() {
    // Object එකක් සාදන්න
    Car myCar("Toyota", "Corolla", 2023);
    
    // Methods භාවිතා කරන්න
    myCar.displayInfo();
    myCar.drive();
    
    return 0;
}
Output:
Brand: Toyota
Model: Corolla
Year: 2023
Toyota Corolla is driving!

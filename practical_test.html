Question 1 Easy
Create a Simple Class for a Student

Write a C++ program to create a Student class with name and age as data members. Create a method to display student information.

// Student class implementation #include <iostream> #include <string> using namespace std; class Student { private: string name; int age; public: // Constructor Student(string n, int a) { name = n; age = a; } // Method to display information void display() { cout << "Name: " << name << endl; cout << "Age: " << age << endl; } }; int main() { Student s1("John", 20); s1.display(); return 0; }

ðŸ“– Description:

  • Class: A blueprint for creating objects. Here, Student is our class.
  • Private members: name and age can only be accessed inside the class.
  • Constructor: Special method that runs when an object is created.
  • Public method: display() can be called from outside the class.
Question 2 Easy
Implement Getter and Setter Methods

Create a Rectangle class with private length and width. Use getter and setter methods to access and modify these values.

#include <iostream> using namespace std; class Rectangle { private: double length; double width; public: // Setter methods void setLength(double l) { if(l > 0) { length = l; } } void setWidth(double w) { if(w > 0) { width = w; } } // Getter methods double getLength() { return length; } double getWidth() { return width; } // Calculate area double getArea() { return length * width; } }; int main() { Rectangle r; r.setLength(5.5); r.setWidth(3.2); cout << "Area: " << r.getArea(); return 0; }

ðŸ“– Description:

  • Encapsulation: Hiding data and providing controlled access through methods.
  • Setter methods: Used to set values with validation (checking if value is positive).
  • Getter methods: Used to retrieve private data safely.
  • Data protection: Direct access to length and width is prevented.
Question 3 Easy
Constructor Overloading

Create a Book class with multiple constructors: default constructor, parameterized constructor with title only, and with both title and author.

#include <iostream> #include <string> using namespace std; class Book { private: string title; string author; public: // Default constructor Book() { title = "Unknown"; author = "Unknown"; } // Constructor with title only Book(string t) { title = t; author = "Unknown"; } // Constructor with title and author Book(string t, string a) { title = t; author = a; } void display() { cout << "Title: " << title << endl; cout << "Author: " << author << endl; } }; int main() { Book b1; Book b2("C++ Programming"); Book b3("OOP Concepts", "John Smith"); b1.display(); b2.display(); b3.display(); return 0; }

ðŸ“– Description:

  • Constructor Overloading: Creating multiple constructors with different parameters.
  • Default constructor: Takes no parameters, sets default values.
  • Flexibility: Users can create objects in different ways based on available information.
  • Same name: All constructors have the same name as the class.
Question 4 Medium
Single Inheritance

Create a base class Vehicle with brand name. Derive a Car class that adds number of doors. Demonstrate inheritance.

#include <iostream> #include <string> using namespace std; // Base class class Vehicle { protected: string brand; public: Vehicle(string b) { brand = b; } void showBrand() { cout << "Brand: " << brand << endl; } }; // Derived class class Car : public Vehicle { private: int doors; public: Car(string b, int d) : Vehicle(b) { doors = d; } void showDetails() { showBrand(); cout << "Doors: " << doors << endl; } }; int main() { Car myCar("Toyota", 4); myCar.showDetails(); return 0; }

ðŸ“– Description:

  • Inheritance: Car inherits properties from Vehicle (parent class).
  • Protected: brand is accessible in derived class but not outside.
  • Constructor chaining: Car constructor calls Vehicle constructor using initialization list.
  • Code reuse: Car can use Vehicle's methods without rewriting them.
Question 5 Medium
Function Overloading

Create a Calculator class with overloaded add() methods that can add two integers, two doubles, and three integers.

#include <iostream> using namespace std; class Calculator { public: // Add two integers int add(int a, int b) { return a + b; } // Add two doubles double add(double a, double b) { return a + b; } // Add three integers int add(int a, int b, int c) { return a + b + c; } }; int main() { Calculator calc; cout << "5 + 3 = " << calc.add(5, 3) << endl; cout << "2.5 + 3.7 = " << calc.add(2.5, 3.7) << endl; cout << "1 + 2 + 3 = " << calc.add(1, 2, 3) << endl; return 0; }

ðŸ“– Description:

  • Function Overloading: Multiple functions with same name but different parameters.
  • Compile-time polymorphism: Compiler decides which function to call based on arguments.
  • Different signatures: Functions differ in number or type of parameters.
  • Convenience: Same operation name for different data types makes code cleaner.

#include <iostream> using namespace std; class Counter { private: static int count; public: // Constructor increments count Counter() { count++; } // Static method to get count static int getCount() { return count; } }; // Initialize static variable int Counter::count = 0; int main() { Counter c1, c2, c3; cout << "Objects created: "; cout << Counter::getCount() << endl; Counter c4; cout << "Objects created: "; cout << Counter::getCount() << endl; return 0; }

ðŸ“– Description:

  • Static variable: Shared by all objects of the class, only one copy exists.
  • Static method: Can be called without creating an object using ClassName::methodName().
  • Initialization: Static variables must be initialized outside the class.
  • Use case: Perfect for counting instances or sharing data across all objects.
Question 7 Medium

Create a Point class with x and y coordinates. Implement a copy constructor to create a copy of an existing Point object.

#include <iostream> using namespace std; class Point { private: int x, y; public: // Parameterized constructor Point(int a, int b) { x = a; y = b; cout << "Parameterized constructor called"; cout << endl; } // Copy constructor Point(const Point &p) { x = p.x; y = p.y; cout << "Copy constructor called"; cout << endl; } void display() { cout << "Point(" << x; cout << ", " << y; cout << ")" << endl; } }; int main() { Point p1(10, 20); p1.display(); // Copy constructor called Point p2 = p1; p2.display(); return 0; }

ðŸ“– Description:

  • Copy Constructor: Creates a new object as a copy of an existing object.
  • Reference parameter: Takes a constant reference to avoid infinite recursion.
  • Automatic call: Called when an object is passed by value or initialized with another object.
  • Deep vs Shallow: Important for classes with dynamic memory allocation.
Question 8 Medium
Friend Function

Create a Box class with private dimensions. Use a friend function to calculate and return the volume of the box.

#include <iostream> using namespace std; class Box { private: double length; double width; double height; public: Box(double l, double w, double h) { length = l; width = w; height = h; } // Declare friend function friend double calculateVolume(Box b); }; // Friend function definition double calculateVolume(Box b) { double volume; volume = b.length * b.width * b.height; return volume; } int main() { Box myBox(3.0, 4.0, 5.0); cout << "Volume: "; cout << calculateVolume(myBox); cout << endl; return 0; }

ðŸ“– Description:

  • Friend Function: Not a member of class but can access private members.
  • Declaration: Declared inside class with 'friend' keyword.
  • Special access: Breaks encapsulation for specific cases where needed.
  • Use case: Useful for operations between two different classes or external operations.
Question 9 Medium
Operator Overloading

Create a Complex number class with real and imaginary parts. Overload the + operator to add two complex numbers.

#include <iostream> using namespace std; class Complex { private: double real; double imag; public: Complex(double r = 0, double i = 0) { real = r; imag = i; } // Overload + operator Complex operator + (const Complex &c) { Complex temp; temp.real = real + c.real; temp.imag = imag + c.imag; return temp; } void display() { cout << real; cout << " + "; cout << imag; cout << "i"; cout << endl; } }; int main() { Complex c1(3.5, 2.5); Complex c2(1.5, 4.5); // Using overloaded + operator Complex c3 = c1 + c2; cout << "c1: "; c1.display(); cout << "c2: "; c2.display(); cout << "c1 + c2: "; c3.display(); return 0; }

ðŸ“– Description:

  • Operator Overloading: Giving special meaning to operators for user-defined types.
  • Syntax: ReturnType operator Symbol (parameters)
  • Natural usage: Allows using + for Complex objects just like built-in types.
  • Common operators: +, -, *, /, ==, <<, >> can all be overloaded.

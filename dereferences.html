Dereferencing

Pointer වලින් Value එක Access කරමු

Dereferencing යනු කුමක්ද?

Dereferencing යනු pointer එකක් භාවිතයෙන් එය point කරන memory location එකේ ඇති value එක access කිරීමයි. මෙය asterisk (*) operator එක භාවිතයෙන් සිදු කරයි.

Dereference Operator (*): Pointer එකක් මඟින් point කරන address එකේ ඇති value එක ලබා ගැනීමට භාවිතා කරයි.
Purpose: Pointer මඟින් memory location එකේ value එක read කිරීමට හෝ modify කිරීමට ඉඩ සලසයි.

Memory Visualization

Dereferencing කොහොමද වැඩ කරන්නේ කියලා බලමු:

Variable Declaration:

Basic Syntax

C++ 
#include <iostream>
using namespace std;

int main() {
    // Variable එකක් declare කරමු
    int num = 100;
    
    // Pointer එකක් create කරලා address එක assign කරමු
    int* ptr = #
    
    // විවිධ ආකාරවලින් access කරමු
    cout << "Value of num: " << num << endl;           // 100
    cout << "Address of num: " << &num << endl;        // 0x... (address)
    cout << "Value of ptr: " << ptr << endl;           // 0x... (same address)
    cout << "Dereferenced ptr: " << *ptr << endl;      // 100 (value at address)
    
    // Dereferencing මඟින් value එක change කරමු
    *ptr = 200;
    
    cout << "\nAfter changing *ptr to 200:" << endl;
    cout << "Value of num: " << num << endl;           // 200 (changed!)
    cout << "Dereferenced ptr: " << *ptr << endl;      // 200
    
    return 0;
}

Asterisk (*) හි භාවිත දෙක

Asterisk (*) operator එක contexts දෙකකදී භාවිතා වෙනවා. අවුල් වෙන්න එපා!

Declaration Context

Pointer type එකක් declare කරන විට:

int* ptr;      // ptr is a pointer to int
double* dptr;  // dptr is a pointer to double
char* cptr;    // cptr is a pointer to char
මෙහිදී * යනු "pointer to" යන්න අදහස් කරනවා
Dereference Context

Pointer එකක value එක access කරන විට:

int num = 50;
int* ptr = #
cout << *ptr;  // Prints 50
*ptr = 100;    // Changes num to 100
මෙහිදී * යනු "value at" යන්න අදහස් කරනවා

ප්‍රායෝගික උදාහරණ

Example 1: Swapping Values Using Pointers
C++ 
#include <iostream>
using namespace std;

void swap(int* a, int* b) {
    // Dereference කරලා values swap කරමු
    int temp = *a;  // temp = value at address a
    *a = *b;        // value at a = value at b
    *b = temp;      // value at b = temp
}

int main() {
    int x = 10, y = 20;
    
    cout << "Before swap: x = " << x << ", y = " << y << endl;
    
    swap(&x, &y);  // addresses pass කරනවා
    
    cout << "After swap: x = " << x << ", y = " << y << endl;
    
    return 0;
}
Example 2: Modifying Array Elements
C++ 
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int* ptr = arr;  // Array name එක pointer එකක් වගේ
    
    cout << "Original array: ";
    for(int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    // Pointer arithmetic සහ dereferencing භාවිතා කරමු
    for(int i = 0; i < 5; i++) {
        *(ptr + i) = *(ptr + i) * 2;  // Each element double කරනවා
    }
    
    cout << "Modified array: ";
    for(int i = 0; i < 5; i++) {
        cout << *(ptr + i) << " ";
    }
    cout << endl;
    
    return 0;
}
Example 3: Working with Objects
C++ 
#include <iostream>
#include <string>
using namespace std;

class Student {
    public:
        string name;
        int age;
        
        void display() {
            cout << "Name: " << name << ", Age: " << age << endl;
        }
};

int main() {
    Student s1;
    s1.name = "Kasun";
    s1.age = 20;
    
    // Object pointer එකක් create කරමු
    Student* ptr = &s1;
    
    // Dereference කරලා members access කරමු
    (*ptr).display();  // Method 1: Dereference කරලා dot operator
    ptr->display();    // Method 2: Arrow operator (වඩා common)
    
    // Members modify කරමු
    (*ptr).age = 21;
    ptr->name = "Kasun Perera";
    
    cout << "\nAfter modification:" << endl;
    ptr->display();
    
    return 0;
}

Reference vs Dereference

Feature Reference (&) Dereference (*)
Purpose Address එක ගන්නවා Value එක ගන්නවා
Syntax &variable *pointer
Returns Memory address Value at address
Use Case Pointer initialize කරන විට Pointer මඟින් value access කරන විට
Example ptr = # val = *ptr;

සාමාන්‍ය වැරදි (Common Mistakes)

Mistake 1: Uninitialized Pointer
// ❌ Wrong - Dangerous!
int* ptr;
*ptr = 10;  // ptr initialize කරලා නෑ - segmentation fault!

// ✅ Correct
int* ptr = nullptr;  // හෝ valid address එකක්
int num = 0;
ptr = #
*ptr = 10;  // Safe now
Mistake 2: Dereferencing NULL Pointer
// ❌ Wrong - Crash!
int* ptr = nullptr;
cout << *ptr;  // NULL pointer dereference කරනවා - crash!// ✅ Correct
int* ptr = nullptr;
if(ptr != nullptr) {
cout << *ptr;
} else {
cout << "Pointer is null!" << endl;
}
Mistake 3: Confusing Declaration and Dereferencing
// ❌ Wrong - Type mismatch
int num = 50;
int* ptr = num;  // Cannot assign int to int*// ✅ Correct
int num = 50;
int* ptr = #  // Assign address
int value = *ptr;  // Get value through dereference
Mistake 4: Dangling Pointer
// ❌ Wrong - Dangling pointer
int* ptr;
{
int num = 100;
ptr = #
}  // num goes out of scope
cout << *ptr;  // Undefined behavior!// ✅ Correct
int* ptr;
int num = 100;  // num has wider scope
ptr = #
cout << *ptr;  // Safe

Advanced Concepts

1. Pointer to Pointer (Double Pointer)

Pointer එකක address එක තවත් pointer එකක store කරන්න පුළුවන්:

C++ 
#include <iostream>
using namespace std;int main() {
int num = 42;
int* ptr = #      // Pointer to int
int** pptr = &ptr;    // Pointer to pointer to intcout << "Value using num: " << num << endl;           // 42
cout << "Value using *ptr: " << *ptr << endl;         // 42
cout << "Value using **pptr: " << **pptr << endl;     // 42// pptr dereference කරනකොට ptr එක ලැබෙනවා
// එය තවත් dereference කරනකොට num එකේ value එක ලැබෙනවා**pptr = 100;  // num එකේ value එක change කරනවා
cout << "After changing **pptr: " << num << endl;     // 100return 0;
}
2. Pointer Arithmetic with Dereferencing
C++ 
#include <iostream>
using namespace std;int main() {
int arr[5] = {10, 20, 30, 40, 50};
int* ptr = arr;// විවිධ ආකාර pointer arithmetic
cout << "First element: " << *ptr << endl;           // 10
cout << "Second element: " << *(ptr + 1) << endl;    // 20
cout << "Third element: " << *(ptr + 2) << endl;     // 30// Increment කරලා access කරමු
ptr++;  // ptr දැන් දෙවන element එක point කරනවා
cout << "After ptr++: " << *ptr << endl;             // 20// Decrement කරමු
ptr--;  // ptr නැවත පළමු element එක point කරනවා
cout << "After ptr--: " << *ptr << endl;             // 10return 0;
}
3. Dynamic Memory with Dereferencing
C++ 
#include <iostream>
using namespace std;int main() {
// Dynamically allocate memory
int* ptr = new int;
*ptr = 75;  // Dereference කරලා value assign කරමුcout << "Dynamically allocated value: " << *ptr << endl;delete ptr;  // Memory free කරමු
ptr = nullptr;  // Dangling pointer වෙන්න දෙන්නේ නෑ// Dynamic array
int size = 5;
int* arr = new int[size];// Array fill කරමු
for(int i = 0; i < size; i++) {
    *(arr + i) = (i + 1) * 10;
}// Array print කරමු
cout << "Dynamic array: ";
for(int i = 0; i < size; i++) {
    cout << *(arr + i) << " ";
}
cout << endl;delete[] arr;  // Array memory free කරමු
arr = nullptr;return 0;
}

Best Practices

Do's
  • Pointer use කිරීමට පෙර initialize කරන්න
  • Dereference කිරීමට පෙර NULL check කරන්න
  • Delete කළ පසු pointer එක nullptr කරන්න
  • Const correctness භාවිතා කරන්න
  • Smart pointers භාවිතා කරන්න (modern C++)
Don'ts
  • Uninitialized pointers dereference කරන්න එපා
  • NULL pointers dereference කරන්න එපා
  • Deleted memory access කරන්න එපා
  • Array bounds එකෙන් පිටත access කරන්න එපා
  • Memory leaks හදන්න එපා

සාරාංශය (Summary)

  • Dereferencing යනු pointer එකක් මඟින් point කරන memory location එකේ value එක access කිරීමයි
  • Asterisk (*) operator එක dereferencing සඳහා භාවිතා කරයි
  • Dereferencing මඟින් pointer වලින් values read සහ write කළ හැක
  • NULL pointers dereference කිරීම program crash කරයි
  • Pointer arithmetic සහ dereferencing එකතු කරලා arrays manipulate කරන්න පුළුවන්
  • Double dereferencing (**) pointer to pointer වලදී භාවිතා වෙනවා
  • Always check pointers are valid before dereferencing
  • Arrow operator (->) object pointers සඳහා වඩා convenient

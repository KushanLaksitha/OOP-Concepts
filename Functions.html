C++ Functions ගැඹුරින්

Virtual, Friend, සහ අනෙකුත් විශේෂිත Functions ගවේෂණය

Virtual Functions

Virtual function යනු base class එකක 선언 කරන ලද සාමාජික ශ්‍රිතයක් වන අතර, එය ව්‍යුත්පන්න පන්තිවල නැවත අර්ථ දැක්විය හැක. මෙය Run-time Polymorphism ලබා ගැනීමට අත්‍යවශ්‍ය වන අතර, base class pointer එකක් හරහා derived class object එකක overridden function එක call කිරීමට ඉඩ සලසයි. `virtual` යන මූල පදය භාවිතයෙන් ශ්‍රිතයක් virtual ලෙස ප්‍රකාශ කරනු ලැබේ.

C++ Virtual Function 
#include <iostream>
using namespace std;
class Animal {
public:
// Virtual function
virtual void makeSound() {
cout << "Animal makes a sound" << endl;
}
};
class Dog : public Animal {
public:
// Overriding the virtual function
void makeSound() override {
cout << "Dog barks: Woof woof!" << endl;
}
};
class Cat : public Animal {
public:
// Overriding the virtual function
void makeSound() override {
cout << "Cat meows: Meow!" << endl;
}
};
int main() {
Animal* animalPtr;
Dog dogObj;
Cat catObj;
animalPtr = &dogObj;
animalPtr->makeSound(); // Calls Dog's makeSound

animalPtr = &catObj;
animalPtr->makeSound(); // Calls Cat's makeSound

return 0;
}
Output:
Dog barks: Woof woof!
Cat meows: Meow!

Friend Functions

Friend function යනු class එකක සාමාජිකයෙකු නොවන නමුත්, එම class එකේ private සහ protected සාමාජිකයන්ට පිවිසීමේ අයිතිය ඇති ශ්‍රිතයකි. `friend` යන මූල පදය භාවිතා කර class එකක් තුළ ශ්‍රිතයක් friend ලෙස ප්‍රකාශ කල හැක. මෙය encapsulation සංකල්පය තරමක් ලිහිල් කරන නමුත්, ඇතැම් අවස්ථාවලදී class දෙකක් අතර සමීප සම්බන්ධතාවයක් අවශ්‍ය වූ විට ප්‍රයෝජනවත් වේ.

C++ Friend Function 
#include <iostream>
using namespace std;
class MyClass {
private:
int private_var;
public:
MyClass() : private_var(10) {}
// Friend function declaration
friend void showPrivate(MyClass& obj);
};
// Friend function definition
void showPrivate(MyClass& obj) {
// Can access private members of MyClass
cout << "The private variable is: " << obj.private_var << endl;
}
int main() {
MyClass myObj;
showPrivate(myObj); // Calling the friend function
return 0;
}
Output:
The private variable is: 10

Inline Functions

Inline function යනු compiler එකට ලබා දෙන යෝජනාවකි. එහිදී, ශ්‍රිතය call කරන සෑම ස්ථානයකදීම function call එකක් සිදු කරනවා වෙනුවට, එම ශ්‍රිතයේ කේතය ආදේශ කිරීමට compiler එකට సూચনা දේ. කුඩා, නිතර call වන ශ්‍රිත සඳහා මෙය භාවිතා කිරීමෙන් function call overhead එක ඉවත් වී program එකේ වේගය වැඩි විය හැක. `inline` යන මූල පදය මගින් මෙය සිදු කරයි.

C++ Inline Function 
#include <iostream>
using namespace std;
inline int add(int a, int b) {
return a + b;
}
int main() {
int result = add(5, 3); // Compiler will likely replace this with '5 + 3'
cout << "Sum: " << result << endl;
}
Output:
Sum: 8

Recursive Functions

Recursive function යනු ශ්‍රිතයක් විසින් එයම නැවත නැවත call කිරීමේ ක්‍රියාවලියයි. මෙම ක්‍රියාවලිය නැවැත්වීමට "base case" එකක් අනිවාර්යයෙන්ම තිබිය යුතුය, එසේ නොමැති වුවහොත් එය අනන්ත පුනරාවර්තනයකට (infinite recursion) ගොස් stack overflow දෝෂයක් ඇති විය හැක. Factorials, Fibonacci ශ්‍රේණිය වැනි ගණිතමය ගැටළු විසඳීමට මෙය බහුලව භාවිතා වේ.

C++ Recursive Function 
#include <iostream>
using namespace std;
long long factorial(int n) {
if (n > 1) {
return n * factorial(n - 1); // Recursive call
} else {
return 1; // Base case
}
}
int main() {
int num = 5;
cout << "Factorial of " << num << " is " << factorial(num) << endl;
return 0;
}
Output:
Factorial of 5 is 120
